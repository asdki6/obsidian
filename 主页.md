## 🚀 进行中的项目
```dataview
TABLE deadline, status
FROM "20 - 项目"
WHERE status = "进行中"
SORT deadline ASC
```

## 📋 散落的任务（所有未完成）
```tasks
not done

```

## 📚 最近更新的资源
```dataview
TABLE file.mtime AS "修改时间"
FROM "30 - 资源"
SORT file.mtime DESC
LIMIT 5
```

---

## 🧠 Wiki 知识库概览
```dataview
TABLE length(rows) AS "页面数"
FROM "wiki"
WHERE type
GROUP BY type
```

### 最近更新的 Wiki 页面
```dataview
TABLE type AS "类型", summary AS "摘要", file.mtime AS "最后更新"
FROM "wiki"
WHERE type
SORT file.mtime DESC
LIMIT 10
```
