# 🏕 freeCodeCamp

> 全球最大开源编程课程

🔗 [GitHub](https://github.com/freeCodeCamp/freeCodeCamp)
🌐 [官网](https://www.freecodecamp.org)

## 为什么没克隆
仓库 576MB，是 TypeScript 全栈工程（课程平台源码），不适合本地学习。直接在官网学习即可。

## 核心内容
- 12 个认证课程（从 HTML 到机器学习）
- 全部免费，支持中文
- 含数千道交互式练习题

## 推荐用法
- [freeCodeCamp.org](https://www.freecodecamp.org/learn) — 直接开始做课程
- 每完成一个模块，在 Obsidian 写一篇总结
- 项目作业完成后链接回笔记
