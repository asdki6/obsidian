# agency-agents

> ⭐ 114,155 | A complete AI agency at your fingertips

## 一句话总结
一个**完整的 AI Agent 协作体系**——从前端向导到 Reddit 社区运营、从创意注入到现实检查，每个 Agent 都是专业角色。

## 核心理念
- **多 Agent 分工协作**：不同 Agent 各司其职
- 每个 Agent 有独立的性格、流程和交付物
- 模拟一个真实"AI 公司"的运作方式

## 与 LLM Wiki 的关联
LLM Wiki 的 Ingest / Query / Lint 三种操作本质上也是**三个 Agent 角色**——各自有专业流程。agency-agents 展示了如何把这种分工体系化。
