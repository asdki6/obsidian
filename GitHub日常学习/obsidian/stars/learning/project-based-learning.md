# project-based-learning

> ⭐ 269,872 | Curated list of project-based tutorials

## 一句话总结
一个精心策划的**项目驱动学习教程合集**，覆盖 Python、JS、Go、C++ 等主流语言，从零构建实际项目来学习编程。

## 核心理念
- 不按教程学语法，而是**通过做项目来学**
- 每个项目都有明确的学习目标和难度分级
- 涵盖网络开发、游戏、CLI 工具、数据库等方向
