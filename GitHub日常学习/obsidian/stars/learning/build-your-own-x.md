# build-your-own-x

> ⭐ 516,709 | Master programming by recreating your favorite technologies from scratch

## 一句话总结
**"自己动手造轮子"**——从零重建 Git、Redis、数据库、编译器、神经网络等核心技术，理解底层原理。

## 核心哲学
- 真正的理解来自**重新实现**
- 不是读源码，而是自己写一遍
- 涵盖：数据库、网络协议、游戏引擎、操作系统、深度学习框架

## 与 AI 时代的学习
有了 LLM 辅助，"自己造轮子"的门槛大幅降低——AI 可以解释架构、提供骨架代码、debug。这个仓库 + LLM = 最高效的底层学习路径。
