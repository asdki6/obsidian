# freeCodeCamp

> ⭐ 449,133 | freeCodeCamp.org's open-source codebase and curriculum

## 一句话总结
全球最大的**免费编程学习平台**——完整课程体系覆盖数学、编程、计算机科学，全部开源。

## 特色
- 全栈课程：HTML/CSS → JS → React → Node.js → Python → ML
- 项目驱动：每个模块以实战项目结业
- 社区驱动：所有课程内容开源协作

## 启示
freeCodeCamp 证明了**结构化课程 + 开源协作**的学习模式——LLM Wiki 也可以朝这个方向演进：从"个人维基"到"AI 辅助的结构化课程"。
