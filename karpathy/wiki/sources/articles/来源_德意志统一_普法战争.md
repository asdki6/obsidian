# 来源_德意志统一_普法战争

## 核心要点

1. **埃姆斯密电**：俾斯麦把普通外交电报删改成挑衅版本——法国舆论炸锅，主动宣战，俾斯麦成功让法国成为"侵略者"
2. **色当战役**：拿破仑三世本人被俘，欧洲史上罕见的皇帝战场被俘，法国主力被歼
3. **凡尔赛宫加冕**（1871.1.18）：在路易十四的镜厅加冕德意志皇帝——对法国两世纪欧洲霸权的终结宣告
4. **阿尔萨斯-洛林的代价**：吞并这块地让法国在未来40年里只有一个国策——复仇，为第一次世界大战埋下种子
