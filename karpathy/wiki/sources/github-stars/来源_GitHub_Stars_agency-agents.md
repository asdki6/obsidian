## 核心要点

1. **多 Agent 分工协作**：前端向导、Reddit 运营、创意注入、现实检查——每个 Agent 有独立性格和流程
2. **模拟"AI 公司"**：不是一个全能 AI，而是一个各司其职的 AI 团队
3. **与 LLM Wiki 操作模式的类比**：Ingest / Query / Lint 三种操作本质上是三个 Agent 角色，各自有专业流程
