# 概念_CS自学体系

## 定义

不依赖正规教育体系，利用开源资源系统性地自学计算机科学的完整方法论。

## 三大支柱

| 支柱       | 代表项目                                      | 作用                 |
| -------- | ----------------------------------------- | ------------------ |
| **学习大纲** | coding-interview-university               | 告诉你"学什么"——完整的知识清单  |
| **路线图**  | developer-roadmap                         | 告诉你"怎么走"——可视化的学习路径 |
| **项目实战** | project-based-learning / build-your-own-x | 告诉你"怎么练"——动手验证理解   |

## 与 LLM Wiki 的结构对应

```
CS 自学体系          →    LLM Wiki 模式
────────────────────────  ────────────────
学习大纲（syllabus）      →   wiki/overview/ 综述页
路线图（roadmap）         →   wiki/index.md 索引
项目实战（project）       →   Ingest → Query → Lint 循环
```

结构化的自学体系和结构化的知识库是同构的——**Wiki 就是你自己写的"学习大纲"**。
