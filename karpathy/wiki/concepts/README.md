# wiki/concepts/ — 概念页

跨来源提炼的抽象概念：方法、理论、模型、框架。

## 命名规范
`概念_名称.md`

## 何时创建
- 一个概念被 2+ 来源提到 → 值得独立成页
- Ingest 时发现新概念 → 创建
- Lint 时发现缺失页面 → 提议创建

## Frontmatter
```yaml
type: concept
tags: [...]
summary: "一句话定义"
sources: ["karpathy/wiki/sources/来源_xxx.md"]
```

## 内容结构
1. 定义
2. 核心原理/步骤
3. 应用场景
4. 在本知识库中的体现
5. 关联实体/概念/来源
