# 概念_AI_Agent学习路径

## 推荐学习链路

```
Python 基础 → LLM 原理 → Agent 入门 → 多 Agent 协作
    │              │            │              │
Python-100-  dive-into-    hello-        agency-
   Days        llms        agents        agents
```

## 各阶段说明

| 阶段 | 项目 | 学什么 |
|------|------|--------|
| 1. 语言基础 | Python-100-Days | Python 编程基础 |
| 2. LLM 底层 | dive-into-llms | Transformer、注意力、预训练、微调 |
| 3. Agent 入门 | hello-agents | Agent 架构、RAG、工具调用 |
| 4. 多 Agent | agency-agents | 多角色分工、协作流程、Agent 团队 |

## 与 LLM Wiki 模式的关联

LLM Wiki 的三步操作本身就是 Agent 的工作模式：
- **Ingest** = Agent 的感知/学习阶段
- **Query** = Agent 的推理/回答阶段
- **Lint** = Agent 的自我修正阶段

学习这个路径不仅是为了"用 AI"，更是为了理解 LLM Wiki 本身的运作机制。
