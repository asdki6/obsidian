# 实体_Obsidian

## 基本信息
- **类型**：软件 / 笔记工具
- **开发者**：Obsidian MD (原 Dynalist 团队)
- **首次发布**：2020
- **官网**：https://obsidian.md

## 特征 / 属性

### 核心技术特性
- **本地优先**：所有笔记以纯 `.md` 文件储存在用户设备上
- **Markdown 原生**：纯文本格式，任何 AI / 编辑器都可读写
- **插件生态**：2700+ 社群插件，覆盖 AI、图表、任务管理等
- **图谱视图**：双向链接可视化，展示知识之间的关联
- **跨平台**：Windows / macOS / Linux / iOS / Android

### AI 相关能力（2026）
- 支持 MCP 协议，Claude Code 等 AI Agent 可直接读写 vault
- 100+ AI 相关社群插件（RAG 对话、行内编辑、多模型切换）
- 支持本地 AI 模型（Ollama、LM Studio）
- 不被任何 AI 供应商锁定

### 数据指标（2026）
- 活跃用户：150 万+
- 年增长：22%
- 平均每日使用：43 分钟
- Sync 价格：$5/月，Publish：$10/月
- 个人使用免费，商用 $50/年

## 关键事件
- 2020：首次发布
- 2024-2025：AI 插件爆发增长（Smart Connections, Copilot, Nova）
- 2025-2026：Obsidian CLI + MCP 支持，AI Agent 原生集成
