## 🧠 核心理解

**函数 = 榨汁机**

- 函数名 → 标签
- 参数 → 投进去的水果
- 函数体 → 刀片和滤网
- 返回值 → 果汁

---

## 📋 参数四个层次

| 层次 | 语法 | 本质 |
|------|------|------|
| 1. 位置参数 | `def f(a, b)` | 按顺序对应 |
| 2. 默认参数 | `def f(a=10)` | 不传走默认 |
| 3. *args | `def f(*args)` | 多余的打包成**元组** |
| 4. **kwargs | `def f(**kw)` | 关键字打包成**字典** |

### 参数顺序法则（必须遵守）

```
位置 → 默认 → *args → 关键字 → **kwargs
```

---

## 💡 关键语法糖

- `/` — 之前的参数**只能按位置传**
- `*` — 之后的参数**只能按关键字传**

```python
def f(a, b, /, *, c):
    pass

f(1, 2, c=3)  # ✅
f(a=1, b=2, c=3)  # ❌ a,b 不能作关键字
f(1, 2, 3)  # ❌ c 必须作关键字
```

## ⚠️ 可变默认值陷阱

```python
# ❌ 危险 — 同一个 list 被反复使用
def bad(item, lst=[]):
    lst.append(item)
    return lst

# ✅ 正确 — 每次新建
def good(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst
```

**原因**: 默认值在 `def` 定义时计算一次，不是每次调用时计算。

## 🔬 实战代码

```python
def 成绩分析器(*分数, 及格线=60, 输出格式="简要"):
    """批量分析考试成绩"""
    if not 分数:
        return {"错误": "无分数"}
    n = len(分数)
    pass_n = sum(1 for f in 分数 if f >= 及格线)
    return {
        "人数": n,
        "平均分": round(sum(分数) / n, 1),
        "及格率": f"{pass_n/n*100:.1f}%"
    }

# 用法
成绩分析器(88, 92, 45, 95, 59)  # 默认及格线60
成绩分析器(88, 92, 45, 95, 59, 及格线=70)  # 自定义
```
