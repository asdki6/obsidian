## README

Master programming by recreating your favorite technologies from scratch.

### Build your own 3D Renderer
- [**Tiny Renderer**](https://github.com/ssloy/tinyrenderer) - Learn how OpenGL works by building a software renderer in ~500 lines of code
- [**Ray Tracing in One Weekend**](https://raytracing.github.io/books/RayTracingInOneWeekend.html) - Learn ray tracing fundamentals

### Build your own Augmented Reality
- [**AR From Scratch**](https://github.com/geohot/arfromscratch) - Build an augmented reality system from scratch

### Build your own Bot
- [**Bot Programming**](https://github.com/topics/bot) - Build a bot for various platforms

### Build your own Blockchain / Cryptocurrency
- [**Naivecoin**](https://lhartikk.github.io/jekyll/update/2017/07/14/chapter1.html) - Build a blockchain in Node.js
- [**Build a Blockchain**](https://medium.com/digitalassetdevelopers/lets-build-a-blockchain-f54f0ce6b5c5) - Build a blockchain from scratch
- [**Go Blockchain**](https://jeiwan.cc/posts/building-blockchain-in-go-part-1/) - Build a blockchain in Go

### Build your own CLI Tool
- [**Build a CLI**](https://www.rust-lang.org/learn) - Build command-line tools in Rust
- [**CLI in Go**](https://gobyexample.com/command-line-arguments) - Build CLI tools in Go

### Build your own Compiler / Interpreter
- [**Tiny Compiler**](https://austinhenley.com/blog/teenytinycompiler1.html) - Build a tiny compiler
- [**Writing a Compiler**](https://norasandler.com/2017/11/29/Write-a-Compiler.html) - Write a compiler from scratch
- [**Make a Lisp**](https://github.com/kanaka/mal) - Build a Lisp interpreter
- [**Crafting Interpreters**](https://craftinginterpreters.com/) - Build a full interpreter

### Build your own Database
- [**Build a Database**](https://cstack.github.io/db_tutorial/) - Build a SQLite clone from scratch
- [**Simple DB**](https://github.com/msiemens/tinydb) - Build a simple database in Python

### Build your own Docker
- [**Docker From Scratch**](https://www.youtube.com/playlist?list=PLbRw3uS3e1MF3PpNY6HlH4w2Nf0Yn0TKp) - Build Docker from scratch
- [**Containers From Scratch**](https://www.infoq.com/presentations/docker-containers-from-scratch/) - Build container technology from scratch

### Build your own Emulator / Virtual Machine
- [**Build an Emulator**](https://www.youtube.com/watch?v=HyznrdDSSGM&list=PLrOv9FMX8xJHfM4oJ4Exf5k8VtA7F2kdQ) - Build a Game Boy emulator
- [**VM From Scratch**](https://www.jmeiners.com/lc3-vm/) - Build a virtual machine

### Build your own Game
- [**Pong in Python**](https://www.pygame.org/docs/tut/PygameIntroduction.html) - Build Pong in Python
- [**2D Game in JavaScript**](https://developer.mozilla.org/en-US/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript) - Build a breakout game
- [**Unity Game**](https://learn.unity.com/) - Build games with Unity

### Build your own Git
- [**Git From Scratch**](https://www.leshenko.net/p/ugit/) - Build Git from scratch
- [**Write Yourself a Git**](https://wyag.thb.lt/) - Write a Git implementation

### Build your own HTTP Server
- [**Simple HTTP Server**](https://www.ics.uci.edu/~ccheng/teaching/tutorials/httpserver/) - Build an HTTP server
- [**Go HTTP Server**](https://gobyexample.com/http-servers) - Build HTTP servers in Go

### Build your own Neural Network
- [**Neural Network From Scratch**](https://www.youtube.com/playlist?list=PLRqwX-V7Uu6aCibgK1PTWWu9by6XFdCfh) - Build a neural network from scratch
- [**Python Neural Network**](https://realpython.com/python-ai-neural-network/) - Build a neural network in Python

### Build your own OS
- [**OS From Scratch**](https://github.com/cfenollosa/os-tutorial) - Build an operating system from scratch
- [**Writing an OS**](https://www.cs.bham.ac.uk/~exr/lectures/opsys/10_11/lectures/os-dev.pdf) - Write a simple operating system

### Build your own Programming Language
- [**Make a Language**](https://www.destroyallsoftware.com/screencasts/catalog) - Build a programming language
- [**Create a Language**](https://medium.com/@marcelogdeandrade/creating-your-own-programming-language-in-python-d93681b09ce1) - Create a language in Python

### Build your own Search Engine
- [**Search Engine**](https://www.analyticsvidhya.com/blog/2021/09/build-a-search-engine-from-scratch/) - Build a search engine from scratch

### Build your own Shell
- [**Write a Shell**](https://brennan.io/2015/01/16/write-a-shell-in-c/) - Write a command-line shell in C
- [**Build a Shell**](https://github.com/danistefanovic/build-your-own-shell) - Build a shell in Python

### Build your own Text Editor
- [**Kilo**](https://viewsourcecode.org/snaptoken/kilo/) - Build a text editor in C
- [**Build a Text Editor**](https://github.com/flutter/samples/tree/master/text_editor) - Build a text editor in Flutter

### Build your own Web Framework
- [**Web Framework in Python**](https://github.com/gregmalcolm/python_web_framework) - Build a web framework in Python
- [**Web Framework in JavaScript**](https://github.com/developit/htm) - Build a web framework in JavaScript

### Build your own Web Server
- [**Web Server in C**](https://github.com/elazarl/simple-web-server) - Build a web server in C
- [**Web Server in Python**](https://docs.python.org/3/library/http.server.html) - Build a web server in Python

### Build your own Web Browser
- [**Browser From Scratch**](https://limpet.net/mbrubeck/2014/08/08/toy-layout-engine-1.html) - Build a web browser from scratch
