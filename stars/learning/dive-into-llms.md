## README

《动手学大模型Dive into LLMs》系列编程实践教程

### 简介

本教程系列专注于大语言模型（LLM）的实践学习，通过编写代码和实验来深入理解大模型的原理和应用。

### 教程大纲

#### 第一部分：大模型基础
1. **Transformer 从零实现**
   - 注意力机制（Attention）
   - 多头注意力（Multi-Head Attention）
   - 位置编码（Positional Encoding）
   - 编码器-解码器架构
2. **预训练语言模型**
   - GPT 系列架构
   - BERT 系列架构
   - T5 模型

#### 第二部分：模型训练
3. **数据准备与预处理**
   - 分词器（Tokenizer）
   - 数据加载
   - 批处理策略
4. **训练技巧**
   - 学习率调度
   - 混合精度训练
   - 梯度累积
   - 分布式训练
5. **微调（Fine-tuning）**
   - 全参数微调
   - LoRA（低秩适配）
   - P-Tuning
   - Prefix Tuning

#### 第三部分：模型推理
6. **推理优化**
   - KV Cache
   - 批处理推理
   - 量化（Quantization）
7. **解码策略**
   - Greedy Search
   - Beam Search
   - Top-K / Top-P Sampling
   - Temperature

#### 第四部分：应用实践
8. **Prompt Engineering**
   - Chain-of-Thought
   - Few-shot Learning
   - Instruction Tuning
9. **检索增强生成（RAG）**
   - 向量检索
   - 文档增强
   - 结果融合
10. **智能体（Agent）**
    - ReAct 模式
    - 工具调用
    - 多步推理

#### 第五部分：模型评估
11. **评估指标**
    - Perplexity
    - BLEU / ROUGE
    - 人工评估
12. **模型压缩**
    - 剪枝
    - 蒸馏
    - 量化

### 环境要求

- Python 3.8+
- PyTorch 2.0+
- CUDA 11.7+（GPU训练推荐）
- Jupyter Notebook / Jupyter Lab

### 特色

- 所有代码可运行，注重实践
- 从原理出发，手写实现
- 配套详细注释和说明
- 中英双语内容
