# Project Based Learning

A list of programming tutorials in which aspiring software developers learn how to build an application from scratch. These tutorials are divided into different primary programming languages. Tutorials may involve multiple technologies and languages.

To get started, simply fork this repo.

### C/C++
- [Build a Compiler](https://austinhenley.com/blog/teenytinycompiler1.html) - Learn how to build a compiler for a simple programming language
- [Build a Text Editor](https://viewsourcecode.org/snaptoken/kilo/) - Build a text editor in C
- [Build a Shell](https://brennan.io/2015/01/16/write-a-shell-in-c/) - Build a command-line shell
- [Build a Game](https://www.youtube.com/playlist?list=PLT7hc3VX1iovFShuOZ3mN5y0N7s1j8Yz) - Build a game in C
- [Build a Web Server](https://github.com/elazarl/simple-web-server) - Build a simple web server in C
- [Build a Ray Tracer](https://github.com/ssloy/tinyrenderer) - Build a ray tracer in C++
- [Build a Physics Engine](https://www.youtube.com/playlist?list=PLW3Zl3wyJwWOpdhYedlDlCBkMl7N9XJmG) - Build a physics engine in C++

### Go
- [Build a Web Application](https://go.dev/doc/tutorial/web-service-gin) - Build a web service in Go
- [Build a CLI Application](https://go.dev/doc/tutorial/create-module) - Build a CLI app in Go
- [Build a REST API](https://github.com/danicat/simple-api) - Build a REST API in Go

### Python
- [Build a Web Scraper](https://docs.python.org/3/library/urllib.html) - Build a web scraper in Python
- [Build a REST API](https://www.restapitutorial.com/) - Build a REST API in Python
- [Build a Game](https://realpython.com/pygame-a-primer/) - Build a game in Python using Pygame
- [Build a Web Framework](https://github.com/gregmalcolm/python_web_framework) - Build a web framework in Python
- [Build a Machine Learning Library](https://github.com/eriklindernoren/ML-From-Scratch) - Build a machine learning library from scratch

### JavaScript/TypeScript
- [Build a Web Framework](https://github.com/developit/htm) - Build a web framework in JavaScript
- [Build a React App](https://reactjs.org/tutorial/tutorial.html) - Build a React app from scratch
- [Build a Node.js CLI](https://nodejs.org/en/docs/guides/nodejs-cli-apps/) - Build a CLI app in Node.js
- [Build a REST API with Express](https://expressjs.com/en/starter/hello-world.html) - Build a REST API with Express

### Full List
The repository contains hundreds of project-based tutorials organized by programming language. Visit the [GitHub repo](https://github.com/practical-tutorials/project-based-learning) for the complete list.
