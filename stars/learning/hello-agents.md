## README

📚 《从零开始构建智能体》——从零开始的智能体原理与实践教程

### 关于本书

本书旨在帮助读者从零开始理解并构建AI智能体（Agent）。从基础概念到高级应用，涵盖智能体的核心原理、设计模式和实战案例。

### 目录结构

#### 第一章：智能体基础
- 什么是智能体
- 智能体的核心组件
- 感知-思考-行动循环
- 环境交互与反馈

#### 第二章：大语言模型基础
- Transformer 架构简介
- 预训练与微调
- Prompt Engineering
- 上下文学习（In-Context Learning）

#### 第三章：智能体架构设计
- ReAct 模式（Reasoning + Acting）
- Plan-and-Solve 模式
- 多智能体协作
- 工具使用与API调用
- 记忆系统设计

#### 第四章：RAG（检索增强生成）
- 向量数据库
- 文档分块策略
- 检索策略
- 生成增强

#### 第五章：工具与框架
- LangChain 入门
- LlamaIndex 入门
- AutoGPT 架构分析
- OpenAI Function Calling
- 自定义工具开发

#### 第六章：多智能体系统
- 智能体通信协议
- 任务分配与调度
- 共识机制
- 案例分析

#### 第七章：实战项目
- 客户服务智能体
- 代码助手智能体
- 数据分析智能体
- 个人知识助手

#### 第八章：部署与监控
- 智能体服务化
- 性能监控
- 日志与追踪
- 安全与对齐

### 技术栈

- **Python** 3.9+
- **LangChain** / **LlamaIndex**
- **OpenAI API** / **开源LLM**
- **ChromaDB** / **FAISS** (向量数据库)
- **FastAPI** (服务部署)

### 前置知识

- Python 基础编程
- 了解基本的机器学习概念
- 对LLM有基本认识
