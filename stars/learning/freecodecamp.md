## README

[![freeCodeCamp.org](https://cdn.freecodecamp.org/platform/universal/fcc_primary.svg)](https://www.freecodecamp.org)

freeCodeCamp.org is a friendly community where you can learn to code for free. It is run by a [donor-supported 501(c)(3)](https://www.freecodecamp.org/donate) nonprofit with the goal of helping millions of busy adults transition into tech.

### What you can learn

Our full-stack web development and machine learning curriculum is completely free and self-paced. We have thousands of interactive coding challenges to help you develop your skills.

**Curriculum:**

1. **Responsive Web Design** - HTML, CSS, Flexbox, CSS Grid
2. **JavaScript Algorithms and Data Structures** - JavaScript, ES6, Regex, OOP, Data Structures
3. **Front End Libraries** - React, Redux, jQuery, Bootstrap, Sass
4. **Data Visualization** - D3.js, JSON, AJAX
5. **Backend Development and APIs** - Node.js, Express, MongoDB, Mongoose
6. **Quality Assurance** - Testing with Chai, Project debugging
7. **Scientific Computing with Python** - Python, Algebra
8. **Data Analysis with Python** - Python, NumPy, Pandas, Matplotlib, Seaborn
9. **Information Security** - HelmetJS, Python security
10. **Machine Learning with Python** - TensorFlow, Neural Networks
11. **College Algebra with Python** - Algebra, Python
12. **A2 English for Developers** - English for coding
13. **Foundational C# with Microsoft** - C#, .NET

### Certifications

Each certification takes approximately 300 hours and includes 5 required projects. All certifications are completely free.

### Tech Stack

- **Frontend**: React, Typescript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Testing**: Cypress, Playwright, Jest
